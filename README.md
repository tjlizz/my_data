
# 历史上的今天

 
```
└─history
    └─10
        ├─21
        └─22
```
根目录为月份,子目录是每天的历史数据。

子目录中的`index.json`文件为当天发生的事情列表，其他为每个事件的详细信息,以`e_id`命名。

# 日历
 
```
└─calendar
    └─10
        ├─21
        └─22
```
根目录为月份,子目录是每天的日历数据,以日期命名。
